What is React?

  • Javascript library (not a framework like Angular) for UI
  • Uses virtual DOM instead of actual DOM
  • "...solve[s] one problem: building large applications with data that changes over time."
source: React Docs

    How do we make components?

    
  // HelloWorld component
  var HelloWorld = React.createClass({
    render: function() {
      return <h1>Hello World!</h1>;
    }
  });

  // To render to page
  ReactDOM.render(
    <HelloWorld />,
    document.getElementById('container')
  );
